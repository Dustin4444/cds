  These components are deprecated and will be removed in a future release. Please migrate to the new chart components: `LineChart` or `AreaChart`.
</Banner>

## Overview

Sparkline components provide compact inline chart visualizations. While still functional, these components are deprecated in favor of the more flexible and feature-rich `LineChart` and `AreaChart` components.

</MDXArticle>
</MDXSection>

<MDXSection>
<MDXArticle>
## Components

- **[Sparkline](/components/graphs/Sparkline/Sparkline/sparkline)** - Basic sparkline visualization
- **[SparklineGradient](/components/graphs/Sparkline/SparklineGradient/sparklineGradient)** - Sparkline with gradient fill
- **[SparklineInteractive](/components/graphs/Sparkline/SparklineInteractive/sparklineInteractive)** - Sparkline with touch/mouse interactions
- **[SparklineInteractiveHeader](/components/graphs/Sparkline/SparklineInteractiveHeader/sparklineInteractiveHeader)** - Header component for interactive sparklines

</MDXArticle>
</MDXSection>

<MDXSection>
<MDXArticle>
## Migration Guide

Replace sparkline components with the equivalent chart components:

```tsx
// Before (deprecated)
import { Sparkline } from '@coinbase/cds-web-visualization';
<Sparkline data={data} />

// After (recommended)
import { LineChart } from '@coinbase/cds-web-visualization';
<LineChart data={data} xAccessor="x" yAccessor="y" />
```
