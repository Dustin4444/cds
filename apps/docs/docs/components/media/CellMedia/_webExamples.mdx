This component will be removed in a future release. Pass media to cells directly via the `media` prop. For example: `<Avatar src={...} />`, `<Icon name={...} />`, `<RemoteImage source={...} />`, or a Pictogram.
:::

### Basic Icon Usage

```tsx live
import { CellMedia } from '@coinbase/cds-web/cells';

<CellMedia active type="icon" name="home" accessibilityLabel="Home icon" />;
```

### Asset Display

Common usage for displaying cryptocurrency assets with proper accessibility.

```tsx live
import { CellMedia } from '@coinbase/cds-web/cells';

<CellMedia type="asset" source={assets.btc.imageUrl} accessibilityLabel="Bitcoin icon" />;
```

### Pictogram Display

Used for displaying pictographic illustrations, commonly used in empty states or feature highlights.

```tsx live
import { CellMedia } from '@coinbase/cds-web/cells';
import { Pictogram } from '@coinbase/cds-web/illustrations';

<CellMedia
  type="pictogram"
  illustration={<Pictogram name="recurringPurchases" dimension="48x48" />}
  accessibilityLabel="Recurring purchases pictogram"
/>;
```
