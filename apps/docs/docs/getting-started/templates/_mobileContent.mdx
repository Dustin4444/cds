## Installation

To create a new project from a template, use `gitpick` to bootstrap your application:

### Expo

```bash
npx -y gitpick coinbase/cds/tree/master/templates/expo cds-expo
cd cds-expo
```

## Setup

After creating your project, install dependencies and start developing:

```bash
# We suggest using nvm to manage Node.js versions
nvm install
nvm use

# Enable corepack for package manager setup
corepack enable

# Install dependencies
yarn

# Start development server
yarn start
```

This will start the Expo development server. You can then:

- Press `i` to open in iOS Simulator
- Press `a` to open in Android Emulator
- Scan the QR code with the Expo Go app on your device

For a full development build with native debugging:

```bash
# iOS (requires Xcode)
yarn ios

# Android (requires Android Studio)
yarn android
```

## What's included

The Expo template comes pre-configured with:

- **Expo SDK 53** with New Architecture enabled (Fabric + TurboModules)
- **React 19** and **React Native**
- Latest CDS packages (`@coinbase/cds-mobile`, `@coinbase/cds-icons`, etc.)
- TypeScript configuration
- Example app demonstrating common UI patterns
- Theme setup with CDS default theme and dark mode support
- Required peer dependencies (`react-native-reanimated`, `react-native-gesture-handler`, etc.)
